# Installation

## Stable release

To install blobchat, run this command in your terminal:

```sh
uv add blobchat
```

Or if you prefer to use `pip`:

```sh
pip install blobchat
```

## From source

The source files for blobchat can be downloaded from the [Github repo](https://github.com/p4prince2/blobchat).

You can either clone the public repository:

```sh
git clone https://github.com/p4prince2/blobchat
```

Or download the [tarball](https://github.com/p4prince2/blobchat/tarball/main):

```sh
curl -OJL https://github.com/p4prince2/blobchat/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd blobchat
uv sync
```
